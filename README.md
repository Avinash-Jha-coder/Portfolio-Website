# 🌐 Avinash Jha – Portfolio Website

## 📌 About the Project

This is my personal **Portfolio Website** built to showcase my projects, skills, certificates, and resume.
The website presents my work as a **Computer Engineering student and aspiring Software Developer**.

It includes sections for **Projects, Certificates, Skills, Resume, and Contact Information**, along with animations and a responsive design.

---

## 🚀 Features

* Modern **dark theme UI**
* Animated sections using **AOS (Animate On Scroll)**
* **Project cards** with GitHub links
* **Certificates section** with preview modal
* **Skill progress bars**
* Embedded **resume viewer with download option**
* Contact section with **social media links**
* Smooth scrolling navigation

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome Icons**
* **AOS Animation Library**
